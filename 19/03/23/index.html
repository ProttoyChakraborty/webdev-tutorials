selectors in CSS (identify which DOM elements/HTML tags to apply the style) 1) Class => applies the style block to a set of elements 2) ID => apply style to only one element ( id needs to be unique )(denoted by #) 3) group => apply the style to all elements having the same tag 4) child => apply the style to children elements
